# Derek Lu - ML Portfolio

Static portfolio website showcasing machine learning systems and optimization projects.

## Live Site

Deploy this repository to [GitHub Pages](https://pages.github.com/) and the site will be available at:

```
https://<username>.github.io/<repo-name>/
```

## Structure

```
.
├── index.html              # Home page (hero, featured projects, categories)
├── ml-systems.html         # ML Systems project cards
├── optimization.html       # Optimization project cards
├── about.html              # Bio, skills, contact links
├── assets/
│   ├── css/
│   │   └── style.css       # All styles, CSS variables, responsive rules
│   └── images/             # Project screenshots (PNG)
└── README.md
```

## Adding Screenshots

Place project screenshot images in `assets/images/`:

| File | Project |
|------|---------|
| `fruit-classifier.png` | Fruit Classifier |
| `stock-multimodal.png` | Multimodal Stock Analysis |
| `fleet-optimization.png` | Fleet Resource Allocation |
| `shipping-scenario.png` | Shipping and Logistics Scenario Analytics |

Recommended image size: 800 x 500 px (16:10 aspect ratio).

## Tech Stack

- Pure HTML + CSS (no frameworks, no build tools, no JavaScript required)
- Google Fonts (Inter)
- Fully responsive and accessible

## Deployment

1. Push this repo to GitHub.
2. Go to **Settings > Pages** and set the source to the `main` branch root.
3. The site will deploy automatically.
